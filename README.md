# Wordkeep / Idiomkeep

英単語・熟語の間隔反復（SRS）学習PWA。単体HTMLファイルで動作し、データは端末内の localStorage にのみ保存されます。

- `index.html` — Wordkeep（単語）
- `idiomkeep.html` — Idiomkeep（熟語・イディオム）。ヘッダーのボタンで相互に切り替え可能

## 機能

- 単語/熟語の追加・検索・編集・削除（重複追加は防止）
- フラッシュカード（暗記用、復習スケジュールには影響しません）
- 4択 / 入力式クイズ。4択の誤答はスペル/意味が似たものを優先的に選出（レーベンシュタイン距離の重み付きサンプリング）
- 間隔反復スケジュール（正解で 1→3→5→7→14→20→30→60→90日後に再出題、不正解で翌日から再スタート）
- Wordkeep: 辞書API（dictionaryapi.dev）＋翻訳API（MyMemory）による単語検索（ログイン不要）。英英辞書の定義も対象言語に翻訳して表示
- Idiomkeep: 入力中に内蔵の熟語データベース（約190個）から候補をオートコンプリート表示。選択すると意味が自動翻訳される。カラーテーマは辞書の表紙をイメージした濃緑
- 両アプリとも、入力中にDatamuse API（無料・キー不要）でオートコンプリート候補を追加表示。
  辞書/熟語データベースに見つからない語には「もしかして」でスペル修正候補（定義付き）を提案
- 多言語対応：日本語 / English / 中文 / Français / العربية / 한국어 / Русский（UI表示と訳の言語を設定で切替、アラビア語はRTL対応）
- テーマ別の単語パッケージ（Wordkeepのみ）：あらかじめ用意した配布デッキを、リスト上部の単語帳セレクターから切り替えて学習。パッケージごとにコンプリート率（%）を表示し、進捗は自分の単語帳（`wordkeep.v2`）とは別キー（`wordkeep.packages.v1`）で完全に分離管理。第1弾は「語学留学の英単語 Part1」（50語）
- 初回起動時のチュートリアル（お試し単語/熟語5つを自動追加）、アップデート後の「更新のお知らせ」、設定 > ヘルプ下に全リリースの更新履歴
- ダブルタップでのズームを無効化（`touch-action:manipulation`）
- JSONバックアップの書き出し・読み込み

## 使い方

`index.html` / `idiomkeep.html` をそのまま開くか、GitHub Pages などで公開するだけで動作します。ビルド手順は不要です。

## データについて

単語データはブラウザの localStorage にのみ保存され、サーバーには送信されません（単語検索時に辞書・翻訳APIへ単語を送る場合を除く）。端末を変えたりブラウザデータを消すと失われるため、設定タブから定期的にバックアップ（JSON書き出し）することを推奨します。

## アクセス解析について

Google Analytics 4（gtag.js）で匿名の利用状況（画面遷移、単語/熟語の追加・削除、復習・クイズの開始/完了と正答率など）を計測しています。`GA_MEASUREMENT_ID`（各ファイル冒頭の`G-XXXXXXXXXX`）を実際の測定IDに差し替えるまでは計測は動作しません。`localhost`/`127.0.0.1`/`file://`など開発環境では常に無効化されます。`app_name`パラメータ（wordkeep/idiomkeep、将来slangkeepも同一プロパティで）で複数アプリを横断比較できます。設定画面にも計測している旨を記載しています。

## AI機能について

以前は [Puter.js](https://puter.com) 経由でカタカナ→英単語のAI推測・例文生成を試みていましたが、Puter側の仕様で初回利用時にログインが必要になることが判明し、実用上使い物にならなかったため撤去しました。ログイン不要で使える無料AI APIが見つかれば再検討します。
